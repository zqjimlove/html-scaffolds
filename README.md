# 前端项目脚手架

## 安装

```sh
npm install -g yo bower git+ssh://git@192.168.0.30:frontend/scaffolds.git
```

## 使用

```sh
mkdir newProject && cd newProject

# 移动端
yo super:mobile 

# 开发
grunt serve

# 编译部署
# 需要设置gruntfile.js的appconfig中的API和CDN

# 编译内网测试版本
grunt build:staging

# 编译外网版本
grunt build:release

```
